# Jujutsu Sync Command

リモートリポジトリから最新の変更を取得し、同期します。

## 手順

1. `jj status` で現在の状態を確認
2. `jj git fetch` でリモートの最新情報を取得
3. `jj log -r 'main..main@origin'` で新しいリビジョンを表示
4. `jj rebase -d main@origin` で現在の作業をリベース

## コンフリクト発生時

- コンフリクトが発生した場合は、対象ファイルを表示
- `jj resolve` でコンフリクトを解決
- ユーザーに解決方法を提案

## 注意事項

- jjでは作業中の変更が自動的に保存されるため、stashは不要
- リベース後は自動的にコンフリクトがあれば表示される
