# お問い合わせフォーム構成案（Googleフォーム用）

## フォームタイトル

**Atsume - お問い合わせ**

## フォームの説明文

```
Atsumeをご利用いただきありがとうございます。
ご質問、ご要望、不具合のご報告など、お気軽にお問い合わせください。
通常3営業日以内にご返信いたします。
```

---

## 質問項目

### 1. お問い合わせの種類（必須・ラジオボタン）

- 不具合・バグの報告
- 機能のリクエスト・改善要望
- 使い方についての質問
- その他

### 2. メールアドレス（必須・メールアドレス形式）

説明文: `ご返信用のメールアドレスをご入力ください`

### 3. アカウントのメールアドレス（任意・メールアドレス形式）

説明文: `Atsumeに登録しているメールアドレス（問題調査に使用します。返信用と同じ場合は省略可）`

### 4. お名前・ニックネーム（任意・記述式）

説明文: `任意でご記入ください`

### 5. ご利用環境（必須・チェックボックス）

- iPhone
- Android
- Web（ブラウザ）

### 6. お問い合わせ内容（必須・段落）

説明文: `できるだけ詳しくご記入ください。不具合の場合は、発生した状況や手順もお書きいただけると助かります。`

### 7. スクリーンショット（任意・ファイルアップロード）

説明文: `不具合の画面などがあれば添付してください`

---

## 送信後のメッセージ

```
お問い合わせありがとうございます。
内容を確認の上、ご返信いたします。
```

---

## Googleフォーム作成時のポイント

1. **回答のコピーを送信**: 設定で「回答者に回答のコピーを送信」を有効にする
2. **スプレッドシート連携**: 回答をスプレッドシートに記録して管理しやすくする
3. **メール通知**: 新しい回答があったときにメール通知を受け取る設定にする
4. **ファイルアップロード**: Googleドライブの容量を使用するため、必要に応じてサイズ制限を設定
