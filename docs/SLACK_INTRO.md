# ATSUME - イベント管理をもっと簡単に

フットサルなどのスポーツイベント主催者向けアプリを作りました！

## 主な機能

- **参加者管理** - 出欠確認・当日チェックイン
- **集金管理** - 誰が払ったか一目でわかる
- **チーム分け** - ランダム or スキル均等で自動分け
- **対戦表** - 総当たり・トーナメントなど5形式対応
- **タイマー** - 試合時間を全員で共有

## 技術スタック

React Native (Expo) / TypeScript / Supabase
